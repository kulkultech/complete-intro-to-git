## Cheatsheet to manage the project

* Create new post

		$ hugo new posts/my-new-post.md

* Run on local

		$ hugo server -D

* Build artifacts

		$ hugo -D

* Publish to GitHub pages

		$ npm install -g gh-pages
		$ yarn global add gh-pages
		$ gh-pages -d public
